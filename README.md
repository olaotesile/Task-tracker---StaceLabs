Interview Task

Task Tracker

A simple task management app built with React Native and Expo.

 Setup
```bash
npm install
npx expo start
```
 Features

- Add, complete, and delete tasks
- Filter by All / Active / Completed
- Tasks persist locally after closing the app

 Libraries Used

- **@react-native-async-storage/async-storage** — for saving tasks locally on the device
- **react-native-web** — to support running the app in a web browser via Expo

 What I'd Improve With More Time
